## 섹션 1: Hero / 서비스 소개

### 배지

RIDY Payout

### 헤드라인

> 정산은 정확하게, 출금은 간편하게.

### 서브카피

> 배달 대행 정산의 새로운 기준, RIDY Payout

### 비전 박스

> **💡 왜 RIDY Payout인가요?**
>
> 복잡한 엑셀 수식과 연락 지옥에서 벗어나세요.
> **관리자**에게는 파일 업로드만으로 끝나는 자동 정산 시스템을,
> **라이더**에게는 기다림 없는 익일 출금 서비스를 제공합니다.
> 플랫폼이 달라도, 인원이 많아도 문제없습니다. RIDY Payout 하나로 정산 고민을 끝내세요.

### 탭 전환 콘텐츠

#### [관리자/지사] 정산 업무가 어떻게 바뀌나요?

- 복잡한 파일 수정 없이, 업로드 한 번으로 정산이 자동 계산됩니다.
- 차감금 기능으로 라이더의 렌트비, 소액 대출까지 누락 없이 관리하세요.
- 지사 프로모션 설정이 정산 내역에 자동으로 반영됩니다.
- 주정산, 익일정산용 이체 파일을 클릭 한 번으로 다운로드하세요.

#### [라이더] 어떤 점이 편해지나요?

- 관리자에게 연락할 필요 없이, 앱에서 직접 '익일 출금'을 신청하세요.
- 매일매일 내 정산 금액과 지급 내역을 1원 단위까지 확인합니다.
- 클릭 한 번으로 플랫폼을 전환해도 내 정산 내역은 그대로 관리됩니다.

## 섹션 2: 주요 기능 — 관리자 (Admin)

### 헤드라인

> 🖥️ PAYOUT Admin (지사용 웹 콘솔)

### 서브카피

> 엑셀 업로드부터 이체까지, 단 3단계로 끝나는 정산 자동화

### 기능 카드

#### 카드 1: 자동 정산 시스템 (Auto-Calculation)

배달 대행 플랫폼의 데이터를 엑셀로 업로드하면, 수수료와 지급액이 **즉시 자동 계산**됩니다.

#### 카드 2: 리스크 & 차감금 관리 (Risk Management)

- **렌트/리스비:** 매일 또는 매주 고정적으로 빠져나가야 할 금액을 간편하게 관리합니다.
- **소액 대출/가불:** 라이더별 남은 부채 금액을 관리하고, 정산금에서 우선 공제하여 미수금을 방지합니다.

#### 카드 3: 지사 프로모션 연동

기상 악화 할증, 프로모션 보너스 등을 설정하면 정산 내역에 **자동 반영**되어 별도 계산이 필요 없습니다.

#### 카드 4: 대량 이체 지원 (Bulk Transfer)

주정산/익일정산 시 은행 양식에 맞춘 **이체 파일 다운로드**를 지원하여, 수백 명의 라이더에게 한 번에 송금할 수 있습니다.

## 섹션 3: 주요 기능 — 라이더 (Rider)

### 헤드라인

> 📱 PAYOUT Rider (라이더용 앱)

### 서브카피

> 내 손안의 정산 비서, 10원 단위까지 투명하게 확인

### 기능 항목

#### 1. 원터치 익일 출금 (Next-day Withdrawal)

매번 지사나 관리자에게 요청할 필요 없이, 앱에서 **'출금 신청'** 버튼만 누르면 접수가 완료됩니다. (매일 자동 신청 설정 가능)

#### 2. 상세 정산 내역 (Dashboard)

단순 총액이 아닌, **수입 - (수수료 + 차감금 + 세금) = 실지급액**의 상세 내역을 영수증처럼 확인할 수 있습니다.

#### 3. 멀티 플랫폼 지원 (Multi-Platform)

여러 배달 대행사를 이용하더라도, 클릭 한 번으로 **플랫폼 계정을 전환**하여 통합 관리가 가능합니다.

## 섹션 4: 이용 가이드 — 관리자

### 헤드라인

> 🛠️ 관리자(Admin) : 빈틈없는 정산 관리

### 아코디언 항목

#### ▶ 1단계: 지사 등록 및 초기 설정

- **기본 정보 등록:** 회원가입 후 [설정] 메뉴에서 운영 중인 지사 정보를 입력합니다.
- **수수료 규칙 세팅:**
  - **기본 공제:** 건당 수수료, 관리비 등 라이더에게 공통으로 적용될 공제 항목을 설정합니다.
  - **원천세 관리:** 3.3% 세금 공제 여부 및 신고 관련 정보를 입력해 둡니다.

#### ▶ 2단계: 차감금 내역 관리

- **차감 리스트 확인:** 라이더별 리스비, 렌트비, 소액 대출 등 차감해야 할 항목을 등록하고 리스트로 관리합니다.
- **관리 편의성:** 시스템이 금액을 강제로 자동 상환(차감)하지는 않지만, 관리자가 정산 시 참고할 수 있도록 **누가, 얼마를 내야 하는지** 명확한 내역을 보여줍니다. 엑셀과 씨름할 필요 없이 화면에서 바로 확인하세요.

#### ▶ 3단계: 정산 파일 업로드 (일간/주간)

- **파일 준비:** 배달 플랫폼 관리자 페이지에서 **'일간 정산 파일'**과 **'주간 정산 파일'**을 다운로드합니다.
- **교차 검증 업로드:** 정확한 정산 데이터를 위해 **두 가지 파일(일간/주간)을 모두 업로드**해야 합니다. 하나라도 누락되면 데이터 정합성을 위해 다음 단계로 넘어가지 않습니다.
- **즉시 확인:** 별도의 '확정' 버튼을 누르거나 라이더에게 알림을 보낼 필요가 없습니다. 업로드가 완료되면 계산된 내역이 즉시 시스템에 반영됩니다.

#### ▶ 4단계: 대량 이체 파일 다운로드

- **표준 양식 제공:** 모든 정산 파일 업로드가 완료되면 **[이체 파일 다운로드]** 버튼이 활성화됩니다.
- **통합 엑셀 파일:** 특정 은행 전용 양식이 아닌, **모든 은행의 대량 이체 시스템에서 요구하는 필수 항목(계좌번호, 금액, 예금주 등)**이 포함된 표준 엑셀 파일을 제공합니다.

## 섹션 5: 이용 가이드 — 라이더

### 헤드라인

> 📱 라이더(Rider) : 간편한 내역 조회

### 아코디언 항목

#### ▶ 1단계: 앱 설치 및 계정 연동

- **설치:** 지사에서 전달받은 **별도의 설치 링크(URL)**를 통해 앱을 다운로드하고 실행합니다. (앱스토어 검색 X)
- **정보 연동:** 소속된 **[지사 코드]**와 사용 중인 **[배달 플랫폼 ID]**를 정확히 입력해야 내 정산 데이터를 불러올 수 있습니다.

#### ▶ 2단계: 카드형 대시보드 조회

- **직관적인 UI:** 보기 편한 **카드(Card)와 리스트(List)** 형태로 내역을 정리해 드립니다.
- **상세 내역:** 오늘 번 수익, 차감된 수수료, 실지급액 등을 스크롤하며 리스트로 간편하게 확인하세요.

#### ▶ 3단계: 익일 출금 신청

- **출금 신청:** 정산 내역 확인 후 [출금 신청] 버튼을 누르면 접수가 완료됩니다.
- **자동 신청:** 매번 신청하기 번거롭다면 [설정]에서 자동 신청 기능을 켜두세요.

### 도입 유형 선택 가이드

#### 🅰️ Type A. 정산 관리만 필요해요 (Basic)

- **기능:** 정산 자동화, 내역 관리, 엑셀/이체 파일 다운로드
- **조건:** 현재 사용 중인 사업자 그대로 유지 (**사업자 이관 X**)
- **추천:** 이미 자금 운용이 원활하지만, 정산 업무(엑셀)가 번거로운 지사

#### 🅱️ Type B. 익일 정산(선지급)도 필요해요 (Premium)

- **기능:** Type A 기능 전체 + **라이더 익일 출금(선지급) 시스템**
- **조건:** 라이더 소속을 'RIDY 파트너스(개인사업자)'로 이관 필요 (**사업자 이관 O**)
- **추천:** 라이더들에게 '매일 정산' 혜택을 주어 경쟁력을 높이고 싶은 지사

### ⚠️ 참고 안내

> **잠깐! '익일 정산(선지급)' 서비스를 이용하려면?**
>
> 라이더에게 정산금을 미리 지급하는 **Type B(프리미엄)** 모델을 도입하려면,
> 라이더의 소속을 **'RIDY 파트너스(개인사업자)'**로 이관하는 절차가 필요합니다.
> *(단순 정산 관리 기능만 사용하는 Type A의 경우, 이관 절차 없이 즉시 사용 가능합니다.)*
